# Установка под Windows

* [Ruby + DevKit](https://rubyinstaller.org/downloads/)

```
gem install gosu
gem install opengl

ruby windows_main.rb
```
# Управление

* WASD - передвижение
* Space - ночной режим
* Esc – выход
* Enter – рестарт
